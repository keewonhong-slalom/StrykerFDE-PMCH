# documentation/

Finished deliverable outputs produced by AI-assisted workflows.

## Organization

Organize outputs by project or engagement:

```
documentation/
  <project-name>/
    <YYYY-MM-DD>-<deliverable-name>.<ext>
```

**Example:**
```
documentation/
  hip-implant-launch/
    2026-04-24-go-to-market-brief.md
    2026-04-24-executive-deck-outline.md
```

## Notes

- All files here are considered finalized or near-final artifacts
- Use dates in filenames to track versions over time
- Source files stay in `raw/` or `reference/` — never here
