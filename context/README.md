# context/

Background information fed to AI agents to ground all outputs in Stryker-specific knowledge.

## Suggested Files to Add Here

| File | What to Put In It |
|------|-------------------|
| `stryker-brand.md` | Brand voice, tone guidelines, messaging pillars |
| `ortho-bu-overview.md` | Business unit structure, key products, strategic priorities |
| `personas.md` | Stakeholder personas for audience targeting |
| `constraints.md` | Any standing constraints—regulatory, legal, confidentiality rules |
| `terminology.md` | Stryker/Ortho-specific terminology and definitions |

## Usage

Reference context files in agent prompts or paste relevant sections into chat before running a workflow.
